## 🔍 풀이

DFS를 활용해서 백트래킹 문제를 풀었다. 
배열에 방문한 숫자 값을 넣어주고 DFS 이후에 빼주었다. 
오름차순 정렬을 위해 for문의 범위는 idx ~ k-1로 해서 순차적으로 돌았다.
