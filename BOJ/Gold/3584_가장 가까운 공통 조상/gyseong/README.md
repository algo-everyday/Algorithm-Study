<a href="https://www.acmicpc.net/problem/3584" rel="nofollow">3584 가장 가까운 공통 조상</a>

## 🔍 풀이
- 루트 노드를 찾고, 루트 노드부터 dfs 시작해서 부모 정보가 들어있는 parent 배열 생성, 찾고자 하는 노드의 아래서부터 탐색하여
첫번째로 겹치는 부분이 공통 조상이다.
